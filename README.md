# Nuevo Vigo360

Versión actual de Vigo360, usando renderizado HTML en Go.

## Requisitos

-   Linux, puede funcionar en otros sistemas
-   Go
-   MySQL
-   Dominio o subdominio apuntando al servidor y certificado TLS
-   Acceso root
